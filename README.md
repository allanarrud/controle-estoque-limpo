# 📦 Sistema de Alerta de Estoque - Monte Sinai Ervas

Projeto desenvolvido para automatizar o controle de estoque da loja Monte Sinai Ervas utilizando Google Sheets como base de dados e envio automático de alertas via WhatsApp.

---

## 🚀 Funcionalidades

- 📊 Leitura automática do estoque via Google Sheets
- ⚠️ Alerta de produtos abaixo do limite mínimo
- ⏳ Aviso de validade próxima (até 10 dias)
- 📱 Envio automático de mensagem via WhatsApp
- 🔐 Autenticação segura com Google Cloud (Service Account)

---

## 🛠 Tecnologias Utilizadas

- Python
- gspread
- pandas
- Google Cloud Platform
- Google Sheets API
- PyWhatKit
- Git & GitHub

---

## 🎯 Objetivo

Reduzir perdas por vencimento e evitar ruptura de estoque, trazendo automação simples e eficiente para um pequeno negócio físico.

---

## 🔐 Segurança

As credenciais da Google Cloud não estão incluídas neste repositório.  
O projeto utiliza autenticação segura via arquivo JSON armazenado localmente.
