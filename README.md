# 📊 Controle de Estoque - Monte Sinai Ervas

Projeto desenvolvido para controle e análise de estoque de produtos naturais, utilizando Google Sheets, Python e MySQL.

## 🚀 Tecnologias
- Python
- MySQL
- Google Sheets
- SQL

## 📌 Funcionalidades
- Registro de entradas e saídas de produtos
- Cálculo automático de estoque
- Controle por unidade (g / un)
- Alertas de estoque baixo via Python (WhatsApp)
- Estruturação de banco de dados relacional

## 🧠 Conceitos aplicados
- Modelagem de dados
- ETL (Extract, Transform, Load)
- Data Quality
- Automação de processos

## 📊 Evolução do projeto
- Fase 1: Controle em Google Sheets
- Fase 2: Automação com Python
- Fase 3: Migração para MySQL (atual)

## 🎯 Objetivo
Transformar dados operacionais em informações estratégicas para tomada de decisão.
